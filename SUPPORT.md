# Support

For product help, setup questions, billing, or MCP client configuration issues:

- Docs: `https://www.regdatalab.com/docs`
- Connect guide: `https://www.regdatalab.com/connect.md`
- Pricing: `https://www.regdatalab.com/pricing`
- Email: `hello@regdatalab.com`

Use this GitHub repo for:

- public docs improvements
- setup/example issues
- lightweight wrapper or metadata issues

Do not use this repo for:

- private account help involving secrets
- billing disputes
- security disclosures

For those, use email instead.
